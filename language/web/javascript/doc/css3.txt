css3模块：
    选择器，盒模型，背景和边框，文字特效，2D/3D转换，动画，多列布局，用户界面

浏览器支持：
    -webkit-或者-moz-，其中前面的数字表示支持该前缀的浏览器的第一个版本


1，边框
    1）圆角边框：
        例子：border-radius:25px;

    2）盒阴影：
        例子：box-shadow：10px,10px,5px #888888;

    3）边界图片：
        使用图像创建边框
        div {
            border-image:url(s1.png) 30 30 round;
        }
